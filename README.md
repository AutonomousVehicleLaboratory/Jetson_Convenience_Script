### Jetson Nano / Jetson Xavier NX initialize
```
cd
git clone https://github.com/FREEWING-JP/Jetson_Convenience_Script
cd
bash ./Jetson_Convenience_Script/JetPack/1st_jetson_initialize.sh
source .bashrc
 or 
sudo reboot
```
